# Audit Checklist (WCAG + UX + Engineering)

## 1. Accessibility (WCAG)

* Contrast warna cukup
* Semua gambar ada alt text
* Navigasi bisa pakai keyboard
* Form ada label jelas

## 2. UX (Impeccable)

* UI konsisten
* Feedback jelas (loading, error)
* Tidak ada dead-end flow
* User tidak bingung navigasi

## 3. Motion

* Animasi tidak berlebihan
* Transisi smooth
* Tidak mengganggu usability

## 4. 7-Lens Audit

* Clarity: jelas?
* Simplicity: simpel?
* Consistency: konsisten?
* Feedback: ada respon?
* Efficiency: cepat?
* Accessibility: inklusif?
* Delight: enak dipakai?

## 5. Layouting

* Grid rapi
* Spacing konsisten
* Responsive (mobile/tablet/desktop)

## 6. Code Quality

* Clean code
* Modular
* No unused code
* Naming jelas

## 7. Performance

* Load cepat
* Asset optimized
* No blocking script

## 8. Security

* Validasi input
* No exposed secret
* Auth aman

## 9. Final Check

* Semua fitur jalan
* Tidak ada error console
* UX smooth end-to-end
